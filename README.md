# 保罗旅行布道 - 策略经营游戏原型

## 游戏简介

一款关于保罗使徒旅行布道的文字策略经营游戏。核心哲学：**在既定的历史轨道（结局固定）中，通过管理有限资源完成使命。**

## 运行方式

### 手动游玩（推荐）

```bash
# 安装依赖
npm install

# 运行游戏
npm start

# 输入 1-4 选择行动：
# 1 - 公开讲道
# 2 - 织帐棚
# 3 - 私下门训
# 4 - 休息
# q - 退出游戏
```

### 演示模式（AI自动游玩）

```bash
npm start -- --demo
```

### 编译后运行

```bash
# 编译 TypeScript
npx tsc

# 运行编译后的 JavaScript
node dist/main.js
```

## 游戏规则

### 核心资源
- **体力 (Stamina)**: 初始 100，用于执行各种行动
- **物资 (Provision)**: 初始 100，维持生存和行动
- **教会健康 (Stability)**: 初始 0，代表所建立教会的成熟程度
- **逼迫指数 (Persecution)**: 初始 0，过高会触发危机事件

### 可选行动

| 行动 | 效果 |
|------|------|
| 公开讲道 | 体力 -20, 教会健康 +15, 逼迫 +20 |
| 织帐棚 | 体力 -15, 物资 +30, 教会健康 -2 |
| 私下门训 | 体力 -10, 教会健康 +5, 逼迫 +2 |
| 休息 | 体力 +20, 物资 -10 |

### 城市路线
1. **安提阿** (3回合) - 逼迫增长率: 3
2. **腓立比** (3回合) - 逼迫增长率: 5  
3. **以弗所** (3回合) - 逼迫增长率: 8

### 危机事件
- **暴动事件**: 逼迫指数 > 70 时触发，损失体力和教会健康
- **分裂风险**: 教会健康 < 30 时触发，教会健康继续下降

### 胜利条件
- 完成所有三个城市的宣教工作

### 失败条件
- 体力耗尽 (< 0)
- 物资耗尽 (< 0)
