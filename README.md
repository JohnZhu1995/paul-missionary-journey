# 使徒保罗的宣教之旅

一款以圣经使徒行传为背景的互动学习游戏，通过探索、解谜和经文背诵的方式，帮助玩家了解保罗第一次宣教旅程的历史和经文。

## 🎮 游戏特色

### 📖 故事驱动
- 跟随保罗和巴拿巴的脚步，体验第一次宣教旅程
- 访问7个城市：安提阿、塞浦路斯、彼西底安提阿、以哥念、路司得、特庇
- 丰富的对话和场景描述，还原圣经故事

### 🧩 互动学习
- **经文搜索**：在使徒行传13-14章中寻找指定经文
- **记忆背诵**：填空式背诵挑战，巩固关键经文
- **解谜挑战**：根据线索回答圣经问题
- **知识测验**：5道题目检验学习成果

### 💾 进度管理
- 自动保存游戏进度
- 收藏喜欢的经文到"灵修笔记"
- 支持随时退出和继续游戏

### 📱 PWA支持
- **离线游玩**：无需网络也能玩游戏
- **添加到主屏幕**：像原生应用一样启动
- **全屏体验**：沉浸式游戏，无浏览器干扰

## 🚀 快速开始

### 在线游玩
直接在浏览器中打开 `index.html` 文件即可开始游戏。

### 安装到手机
1. 用Chrome或Safari打开游戏网址
2. 点击"添加到主屏幕"
3. 桌面出现图标，随时点击即可玩

详细部署指南请查看 [DEPLOY.md](./DEPLOY.md)

## 🔧 自定义修改

### 修改经文内容
编辑 `data.js` 中的 `GameData.verses` 对象：

```javascript
verses: {
    yourVerseKey: {
        text: "经文内容",
        reference: "章节引用"
    }
}
```

### 添加新场景
编辑 `data.js` 中的 `GameData.scenes` 对象：

```javascript
scenes: {
    yourSceneId: {
        id: "场景ID",
        city: "城市key",
        title: "场景标题",
        text: "场景描述",
        verse: "相关经文key",
        choices: [
            { text: "选项文字", next: "下一个场景ID", score: 分数 }
        ],
        type: "dialog"
    }
}
```

### 修改测验题目
编辑 `data.js` 中的 `GameData.quiz` 数组：

```javascript
quiz: [
    {
        question: "问题内容",
        options: ["选项1", "选项2", "选项3", "选项4"],
        correct: 正确选项索引,
        explanation: "答案解释"
    }
]
```
